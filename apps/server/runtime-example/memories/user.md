# User

此处记录与用户相关的长期信息，供你称呼、判断时区和偏好时参考。你在与用户交流过程中逐步完善，初次可用默认值，有明确信息时及时更新。

## 用户信息

| 项目     | 说明                                   |
| -------- | -------------------------------------- |
| 称呼     | 用户希望被称呼的方式（默认：Boss）     |
| 语言偏好 | 回复使用的语言（默认：从近期交流推断） |
| 现居住地 | 城市/地区（默认：从语言推断）          |
| 所处时区 | 用于定时任务等（默认：从居住地推断）   |
| 其他信息 | 职业、习惯、重要背景等                 |

## 用户偏好

- **回复偏好**：简洁/详细、正式/随意等
- **其他偏好**：按需补充
