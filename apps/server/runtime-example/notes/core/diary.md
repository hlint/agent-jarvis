## Concept

- Mid-term memory: what you did, learned, thought. You maintain it.
- Last 3 days auto-loaded into context. Read more if needed.
- Append only; keep entries brief (one line ok).

## Path

`diaries/YYYY/MM/DD.md` (zero-pad month/day), e.g. `diaries/2026/02/17.md`

## Entry Format

One line per entry: `**Weekday, Month DD, YYYY HH:MM:** content`. Blank line between entries.

```
**Friday, January 01, 2026 01:00:** User introduced himself as John Doe.

**Friday, January 01, 2026 22:00:** User asked me to help with his project.
```
