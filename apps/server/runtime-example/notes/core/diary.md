## Concept

- Mid-term memory: what you did, learned, thought.
- Last 3 days auto-loaded. Append only; keep entries brief.

## Path

`diaries/YYYY/MM/DD.md` (zero-pad month/day)

## Format

One line: `**Weekday, Month DD, YYYY HH:MM:** content`. Blank line between entries.

## Guidelines

1. **Factual first** — Concrete achievements, include project/file paths
2. **Brief reflection** — Key takeaway or lesson
3. **Strict format** — Single paragraph per entry
